# 🌙 Eid Greeting Website

[![Live Demo](https://img.shields.io/badge/Live-Demo-green?style=for-the-badge)](https://username.github.io/eid-greeting)

Website ucapan **Selamat Hari Raya Idul Fitri** dengan dekorasi langit malam, animasi teks, dan musik latar.

Project ini dibuat menggunakan **HTML, CSS, dan JavaScript** tanpa menggunakan framework.

## ✨ Features

- Animasi teks ucapan bertahap
- Musik latar dengan tombol **mute / unmute**
- Dekorasi langit malam
  - 🌙 Moon
  - ⭐ Stars
  - 🏮 Lanterns
  - 🕌 Mosque silhouette
  - 🟩 Ketupat animation

- Responsive design (Desktop, Tablet, Mobile)

---

## 🛠 Tech Stack

- HTML5
- CSS3
- Vanilla JavaScript

## 🎯 Purpose

Project ini dibuat sebagai latihan **Front-End Web Development** untuk mempelajari:

- Layouting dengan CSS
- Animasi CSS
- Manipulasi DOM menggunakan JavaScript
- Interaksi user (button & audio)
- Responsive web design
